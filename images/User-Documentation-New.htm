User Documentation for New OpenCPN user

Requirements: To use this plugin you must have OpenCPN running and a good internet connection that is active. Since you will be downloading megabytes of charts it is best to have a good connection. Poor or interrupted internet connections will create more trouble messages.

Download the Plugiin: You will find the plugin available for download here where you will find instructions for your OS.After downloading and installing the plugin, restart OpenCPN and select Options > Charts > Chart Downloader Tab, which is the plugin. There will be no icon.

1-Chart Downloader Tab

Overview: This screen has two sections.

The top section is for Chart Catalogs which are retrieved from sources on the internet using the "Add" button. Once the "Add" button settings for source Catalog and destination directory have been accepted the Catalog will appear in the upper window. Selection of "Update" button will prompt the new Catalog to query for a current List of Charts (xml file) which will appear in the lower window.

The bottom section has the "List of Charts" for the highlighted "Chart Catalog" in the upper window. Specific charts can be selected for "Download" by checking the boxes or right moust click in the bottom window area and selecting an appropriate action.

Let's step through this to see how it works.

.

2 Add New Predefined Chart Source

Select "Add" button, then select a "Predefined" Catalog from the Catalog dropdown menu.You may scroll to the bottom and observe, but for learning purposes, pick thee first one "US RNC Region 02". Hit enter to select.

.

3 OK Predefined Chart Directory Set Automatically

Since the "Predefined" radio button is selected, the Name and URL and destination chart directory are predetermined. Just continue with "OK". Note that a "..Charts/RNC/US_Region 02" directory is created. For Windows users, the predefined Chart directory will be under ProgramData/OpenCpn adjacent to other Opencpn data.

.

4 OK Predefined Chart Directory Set Automatically

This is a closer view of the menu for better inspection.

.

5 Catalog Entered

Here the "RNC_Catalog_02_Region.." has been entered into the top window. The catalog "Add" date is shown, and by pulling out the Options window you can see the full destination pathname. Next you must highlight the Catalog, so that "Update" button becomes active.

.

6 Chart List Updated and Charts checked

When the "Update" Button to the right of the upper window is selected, the Chart List below will be freshly populated. window. Select the top three charts for "Newport", "Approaches to New York" and "Cape Sable". Hit "Download Selected Charts" Now several download windows may pop up as these charts are downloaded.

.

7 Downloading Charts

Downloading...multiple charts can take quite awhile.

.

8 Downloading Charts

Downloading can be as simple as that with Chartdldr, however you will find that there is more to learn about downloading and using charts, because you need to setup Directory pointers to your chart folders and to define Chart Groups and add folders to the groups. This is described in the OpenCPN manual more completely.

.

9 Three charts downloaded with "Show local files" Menu

Here the first three charts are shown "Up to Date" and the button "Show Local Files" has been pressed to cause the Destination Folder to open in a separate window.

Right Mouse Click in Chart Window will open a selector window with choices for "Select all","Select none", "Select new", "Select update", "Invert selection". The user can use these tools as appropriate.

We have downloaded three "US_RNC_Region_02" (raster) charts by "Adding" the Region 02 Catalog to the top window, "Updating" it to show the List of Charts below, then selecting some charts and "Downloading". Now we are going to do the same thing for US_ENC_Region_02 Charts (Vector charts) again, "Adding" a new Catalog, using "Predefined" radio button.

.

10 ENC Add New Predefined Chart Source

Hit "Add" in the upper window area, select US_ENC_Region_02 from the dropdown Catalog menu. Again the Name, URL and Chart Directory are Predefined. You just need to hit "OK". Then highlight "US_ENC_Region_02" and hit "Update" and the new charts in that catalog appear in the lower window.

.

11 ENC Chart List Updated Charts checked

Now select several charts with your mouse or by using right click. Then select "Download selected Charts" and the Chartdownloader will start progressively downloading. If there is a chart which fails, uncheck it, download and theen try downloading the problem chart.

.

12 Download Selected Charts

Downloading .... If there is a chart which fails, uncheck it, restart download and then try downloading the problem chart.

13 Add New Custom Catalog

"Add" new "Custom" (radio button) Chart Catalog. Note that the Name, URL and Chart directory are completed manually. The browse window for selection of the destination directory is shown. The rest of the custom operation is very similar to "predefined" operation, however note that the URL and Destination directories have to be selected very carefully.
